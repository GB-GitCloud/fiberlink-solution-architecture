# Plataforma de Integración Empresarial

## Objetivo

Implementar una plataforma centralizada de integración basada en APIs y eventos que elimine las integraciones punto a punto y permita la interoperabilidad entre aplicaciones legacy, SaaS y nuevas aplicaciones.
