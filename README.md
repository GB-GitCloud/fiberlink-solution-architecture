# Objetivo

Diseñar la Arquitectura Empresarial y la Arquitectura de Solución para la modernización tecnológica de Fiberlink Telecom, aplicando TOGAF ADM, C4 Model y principios Cloud Native.

---

# Caso de Negocio

Fiberlink Telecom busca modernizar su plataforma tecnológica para reducir tiempos de provisión, mejorar la experiencia del cliente, incrementar la automatización operacional y habilitar capacidades analíticas para la toma de decisiones.

---

# Alcance

El proyecto comprende:

- Arquitectura Empresarial (TOGAF ADM)
- Arquitectura de Solución
- Arquitectura Multinube
- Arquitectura de Integración
- APIs
- C4 Model
- ADR (Architectural Decision Records)
- Requerimientos Funcionales
- Requerimientos No Funcionales

---

# Tecnologías

- Azure

- TOGAF ADM
